# Landing ULTRAFACILE (ultrafacile.fr)

Page d'accueil de l'agence ULTRAFACILE. Site statique, un seul fichier `index.html` autonome (CSS + JS inline), images locales.

## Contenu
- `index.html` : la page (police Outfit via Google Fonts, smooth scroll Lenis via CDN).
- `cases/` : visuels d'illustration des cas clients.
- `Sabine.jpg`, `anthony.jpeg`, `Emi.jpg` : portraits et témoignage.

## Déploiement
Site 100% statique. Uploader le contenu de ce dossier à la racine web de l'hébergement (Hostinger `public_html`), ou servir via GitHub Pages.

## Coût d'exploitation
Aucun coût d'hébergement propre (statique). Dépendances externes gratuites : Google Fonts, CDN Lenis. La prise de rendez-vous passe par TidyCal (offre TidyCal de Sabine).
